プライバシーポリシー

1. 個人情報の収集・利用目的

アプリの機能提供および改善、ユーザーサポート、広告配信のために個人情報を収集します。

2. 収集する情報

  • 学習データ（単語、学習履歴等）
  • デバイス情報
  • 広告IDおよび関連情報

3. 第三者サービスの利用について

本アプリでは以下の第三者サービスを使用しています：

  • Google AdMob（広告配信）

    広告配信のために広告IDを使用します。詳細はGoogleプライバシーポリシーをご確認ください。

  • OpenAI ChatGPT API

    AI機能提供のために入力された単語データを使用します。詳細はOpenAIプライバシーポリシーをご確認ください。

4. 個人情報の管理

適切な安全管理措置を講じ、法令に基づく場合を除き第三者への提供は行いません。

5. アクセス解析

利用状況の分析のため、匿名データを収集します。これらは個人を特定しない形で統計的に処理されます。

6. プライバシーポリシーの変更

本ポリシーは予告なく変更される場合があります。変更時は本アプリ内で通知します。

利用規約

1. 利用規約の適用

本規約は、本アプリの利用に関する条件を定めるものです。ユーザーは本規約に同意した上でアプリを利用するものとします。

2. サービス内容

  • 英単語学習支援アプリとして以下を提供します：
    • 単語登録・管理機能
    • 復習機能
    • 学習進捗管理機能
  • AIによる例文・意味の生成機能を含みます
  • アプリ内に広告を表示します

3. 禁止事項

  • 不適切な単語の検索・登録
  • システムへの過度な負荷
  • 他のユーザーへの迷惑行為
  • 法令違反行為
  • その他、当社が不適切と判断する行為

4. 免責事項

  • AIによる生成内容の正確性は保証されません
  • サービスの停止・変更がある場合があります
  • 本規約は予告なく変更される場合があります
